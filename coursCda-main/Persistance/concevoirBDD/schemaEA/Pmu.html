Contexte

Les règles de gestion sont les suivantes : • un cheval est identifié par ses caractéristiques de naissance : • un numéro d'enregistrement auprès de la société des courses, • une date, • une mère toujours identifiée, • une race, • une couleur, • un sexe, • un lieu de naissance • un cheval est la propriété, à une date donnée, d'un propriétaire ou d'un ensemble de propriétaires regroupés en société. • un cheval est entraîné, à une date donnée, par un entraîneur qui peut lui-même être propriétaire. • le propriétaire (ou l'entraîneur) est identifié par un numéro, une adresse (rue, ville, code postal). • un cheval est suivi par un vétérinaire, chargé à des dates données d'établir ses vaccinations

partie 2

Pour le programme de COURSES de CHEVAUX en région parisienne d'autres éléments d'informations extraits de la phase de recueil sont les suivants : • il existe 6 champs de courses en région parisienne. • il est prévue au maximum deux réunions de courses par jour (après-midi et nocturne). • le nombre de chevaux par course est fixé au minimum à 5 chevaux et au maximum à 25 chevaux. • Le programme des courses à établir concerne une semaine quelconque dans l'année. • Les informations nécessaires pour chaque course sont les suivantes : ª le numéro de la course, ª le nombre de partant, ª la distance, ª le prix pour le gagnant, ª l'heure de départ; ª le nom de l'épreuve, ª le type de course (trot, galop, obstacle). • Les informations nécessaires pour un partant sont les suivantes : ª le numéro du cheval dans la course, ª le nom du cheval, ª le nom du père et de la mère, ª le nom du propriétaire, ª le nom de l'entraîneur, ª le nom du jockey ou du driver, ª la place à la corde, ª le handicap (poids ou distance), ª les gains du cheval, • les couleurs du propriétaire. • Les éléments d'information suivants ont été fournis : ª Un jockey ou un driver peut monter plusieurs fois dans la même réunion de courses. ª Un cheval peut réaliser plusieurs courses dans la même semaine mais pas le même jour. ª Un cheval ne peut courir que dans un seul type de course ª Un propriétaire ou un entraîneur peut avoir plusieurs chevaux dans la même épreuve (écurie).
